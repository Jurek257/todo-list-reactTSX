# ToDo Application

Eine Anwendung, die zum Lernen von React + TypeScript erstellt wurde.

## Installation und Start
```bash
# Abhängigkeiten installieren
npm install

# Im Entwicklungsmodus starten
npm run dev
```

Die Anwendung ist live verfügbar: [https://todo-list-react-tsx.vercel.app/](https://todo-list-react-tsx.vercel.app/)

## Technologien

- React 18
- TypeScript
- Vite (Build-Tool)
- CSS3
- localStorage

## Funktionalität

- Neue Aufgaben hinzufügen
- Aufgaben als abgeschlossen markieren
- Aufgaben entfernen
- Zähler für aktive Aufgaben
- Daten speichern (localStorage)
- Alle Aufgaben entfernen
